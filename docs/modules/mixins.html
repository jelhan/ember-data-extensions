mixins Module

Defined in: packages/mixins/lib/embedded_json_mixin.js:11

The EmbeddedJSONMixin allows you to add embedded record support to your serializers.

To set up embedded records, you include the mixin into the serializer and then define your embedded relations. The EmbeddedJSONSerializer is an example.

Below is an example of a per type serializer (post type).

App.PostSerializer = DS.RESTSerializer.extend(DS.EmbeddedJSONMixin, {
  attrs: {
    author: {embedded: 'always'},
    comments: {embedded: 'always'}
  }
})

Currently only {embedded: 'always'} records are supported.
